Welcome to Ampt

This page is displayed when you visit your sandbox URL before building your app.

To see your app go to the Ampt CLI and run one of the following commands:

  • run the "dev" command to start the dev server and visit the localhost URL that is displayed in the CLI, or
  • run the "build" command to build your app and refresh this page

You can remove this page by deleting the "welcome.html" file in the project, and removing the code that displays it in the "index.js" file.
